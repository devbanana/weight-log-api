# Weight Log API

A smart weight and calorie tracking API designed to help you reach your health goals through data-driven insights. This backend application provides comprehensive statistics, trend analysis, and progress tracking capabilities for the Weight Log frontend ([weight-log-web](https://github.com/devbanana/weight-log-web/)). Built with Symfony 7.3, PHP 8.4, and following strict Domain-Driven Design and Hexagonal Architecture principles, this API emphasizes clean architecture, rich domain models, CQRS pattern, and comprehensive testing to ensure reliability and maintainability. Features include JWT-based authentication, RESTful endpoints with OpenAPI documentation via API Platform, and PostgreSQL/MySQL database support through Doctrine ORM.
